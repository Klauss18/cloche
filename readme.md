#  [![NPM version][npm-image]][npm-url] [![Dependency Status][daviddm-url]][daviddm-image]

> Show/Hide all files on desktop on MacOS


## Install

```sh
$ npm install -g cloche
```


## Usage

```sh
$ cloche show
$ cloche hide
$ cloche --help
```
